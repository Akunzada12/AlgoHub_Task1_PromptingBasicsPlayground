# Prompting Basics Playground

## Overview
This project demonstrates Zero-Shot Prompting using Google's Gemini AI.

## Features
- Enter any prompt
- Generate AI responses
- Compare different prompt styles

## Technologies Used
- Python
- Streamlit
- Gemini API

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
py -m streamlit run app.py
```
